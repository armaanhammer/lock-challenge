41 ## Design Constraints
42 
43 * main()
44  * adds command handlers in Controller class to CommandDispatcher using addCommandHandler
45  * invokes CommandDispatcher::dispatchCommand using a raw string.
46 * CommandDispatcher()
47  * stores command handlers in Controller class using a private map
48  * invokes command handler functions using either rapidjson::value or rapidjson::document objects
49 * Controller()
50  * implements command handler functions

52 ## Design Decisions
53 
54 * main()
55  * prints success to user
56  * catches failure messages exceptions and prints to user
57 * CommandDispatcher()
58  * parses raw string from main into a rapidjson::document
59  * performs initial checks to verify that "command" and "payload" values exist
60  * extracts "payload" value from document and creates new document to pass to command handler
61  * attempts to invoke command handler function matching "command"
62  * prints success to user
63  * passes failures as exceptions back to main
64  * passes exceptions from Controller back to main
65 * Controller()
66  * performs commanded functions
67  * functions check whether their individual parameters are satisfied.
68  * prints success to user
69  * passes failures as exceptions back to CommandDispatcher
