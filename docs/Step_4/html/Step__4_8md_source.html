24 ### Coding
25 
26 I chose `cout` as my common resoure to protect. To accomplish this, I only need to lock a mutex and call my `chal::LockGuard` function from within the `thd_printer` again.
27 
28 I debated about this for a bit. See [Reference in Appendix](Step_4_Appendix.md#) for details. In the end, I decided to enable DEBUG printing inside of the `chal::LockGuard` class to demonstrate that locking and unlocking are occuring with each write to standard out from the threads.
